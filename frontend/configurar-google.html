Configurar Google Sign-In

Sigue estos pasos una sola vez. Luego el botón de Google funcionará con tu cuenta real.

  1. 1

    Abre Google Cloud Console

    Inicia sesión con tu cuenta de Google.

    Abrir Credenciales →
  2. 2

    Pantalla de consentimiento (si es la primera vez)

    Tipo Externo, nombre de app Odontolyt. En Usuarios de prueba añade tu Gmail.

    Pantalla de consentimiento →
  3. 3

    Crear ID de cliente OAuth

    Crear credenciales → ID de cliente de OAuth → Aplicación web.

    En Orígenes de JavaScript autorizados agrega exactamente:

    http://localhost:3000
  4. 4

    Pega tu Client ID aquí

    Copia el ID que termina en .apps.googleusercontent.com

Mientras no configures el Client ID, puedes usar el botón de Google en modo demostración en inicio de sesión.
